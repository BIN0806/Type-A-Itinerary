# Documentation

This directory contains all project documentation organized by category.

## Structure

### `set-up/`
Setup and configuration guides:
- `QUICKSTART.md` - Quick start guide for getting the project running
- `OPENAI_API_SETUP.md` - OpenAI API key setup and troubleshooting
- `README.md` - Main project README
- `mobile-README.md` - Mobile app specific documentation

### `bug-fixes/`
Documentation of bug fixes and troubleshooting:
- `CONFIG_FIX_SUMMARY.md` - Configuration validation fixes
- `TDD_FIX_SUMMARY.md` - Account registration TDD fix
- `IMAGE_UPLOAD_TDD_FIX.md` - Image upload issue fixes

### `implementation/`
Implementation details and specifications:
- `ENHANCED_PIPELINE_IMPLEMENTATION.md` - Enhanced image scanning pipeline
- `IMPLEMENTATION_SUMMARY.md` - Overall implementation summary
- `SPEED_OPTIMIZATION.md` - Performance optimization notes
- `V2V_PROJECT_SPECS.md` - Project specifications
- `TEST_PLAN.md` - Test planning documentation
- `NEXT_STEPS.md` - Future work and next steps
